## 1. 项目简介

本项目进行了：量化策略复现工作 [参考研报](#6-代码来源与引用reference)。

基于Carhart 四因子模型（市场、市值、估值、动量），我将个股的**周末特质波动率 - 周内特质波动率**作为因子，并以**月度调仓**的思路，构建多空组合并进行回测评估。
$$R_{i,t} = \alpha_i + \beta_{i,MKT} MKT_t + \beta_{i,SMB} SMB_t + \beta_{i,HML} HML_t + \beta_{i,UMD} UMD_t + \epsilon_{i,t}$$
$$Factor_{i,t} = \epsilon_{i,weekend} - \epsilon_{i,weekday}$$

项目使用 Python（pandas、statsmodels、matplotlib 等）实现，**代码与数据完整可运行**，并给出关键回测指标与图表。

## 2. 策略原理
在得到部分印证的半强有效市场假说下，股票价格反映了所有公开信息，但存在反应不足的情况。每天都有新的信息出现，但每一天的信息质量是不同的。

**理论逻辑**：周末相对于工作日的信息质量更高，因为周末的信息获取成本更高。具体到横截面上，每只股票的信息质量存在差异，通过周一的收益率与周二至周五的收益率差值能很好地刻画每只股票周末的信息质量。

**计算逻辑**：周一**特质波动率**高，代表这只股票的周一的收益率中存在较多无法被模型解释的部分。这可能来自于**质量高、市场反应积极**的信息。市场会在未来缓慢反应这个信息，由此为我们带来了超额收益。
1. **数据频率与聚合**：
   - 以 Tushare 的日线数据为基础，按交易周（`W-FRI`）聚合，得到**每只股票**与**基准指数**的**周一收益与周内其他时间收益**。

2. **定价因子计算**：
   - 对齐个股与指数的同期收益
   - $SMB$ 上一月底的流动市值排名后三分之一股票组合(即小市值股票组合)的收益减去流动市值排名前三分之一股票组合(即大市值股票组合)的收益。
   - $HML$ 上一月底按账面市值比前三分之一股票组合(即低市净率股票组合)的收益减去账面市值比排名后三分之一股票组合(即高市净率股票组合)的收益
   - $UMD$ 上一月底按当月累积收益排名前三分之一股票组合(即高 1 个月动量股票组合)的收益减去当月累积收益排名后三分之一股票组合(即低 1 个月动量股票组合)的收益。

3. **回归取残差**：
   - 将每只股票的周一收益、周内其他时间收益分别对 **市场（基准指数同期收益）**、**市值（SMB）**、**动量（UMD）**、**估值（HML）** 做 OLS 回归
   - **具体算法为**：
   - 将定价因子合并到一张表df_pricing_merge，（只在内存，不离线）结构为 `[trade_date, MKT, SMB, HML, UMD]`，每一行对应每个交易日
   - 使用Jolib，进行多进程计算。按照每一只股票分组，依据个股历史的周一收益、周内其他时间收益$Y_{i,weekend}$、$Y_{i,weekday}$，将定价因子merge到个股中，求得
   $$\epsilon_{i,weekend} = Y_{i,weekend} - \beta_{weekend}Y_{i,weekend}$$
   $$\epsilon_{i,weekday} = Y_{i,weekday} - \beta_{weekday}Y_{i,weekday}$$
   - 将每只股票周一回归的残差$\epsilon_{i,weekend}$ 减去 周内其他时间的回归残差 $\epsilon_{i,weekday}$，即为最终因子
   $$Factor_{i,t} = \epsilon_{i,weekend} - \epsilon_{i,weekday}$$

4. **投资组合构建与回测**：
   - 月度调仓
   - 将当月每周的因子值取平均，作为该月因子值
   - 将股票按因子值分为 `N=10` 组，做 **最高组多头、最低组空头** 的等权多空组合。
   - 计算 **IC/IR**、**分组收益**、**多空（L–S）组合收益**、与**基准收益**，并给出**夏普比率**、**累计收益曲线**等。

> 备注：该思路运用经典的资产定价模型，将计算的特质波动率差异作为因子。对于格式统一的金融数据，计算过程体现了极具针对性的编程方法与数据处理能力。
